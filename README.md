## Instalação

### Clone o projeto
```
git clone https://github.com/gileno/djangoecommerce
cd djangoecommerce
```

### virtualenv
```
virtualenv env -p python3
```
Linux
```
source env/bin/activate
```
Windows
```
env\Scripts\activate.bat
```



## Instale as dependencias
```
pip install -r requirements.txt
python manage.py runserver
```
